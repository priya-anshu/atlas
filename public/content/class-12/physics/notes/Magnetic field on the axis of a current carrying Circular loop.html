Magnetic Field on the Axis of a
Current-Carrying Circular Loop

Full derivation using Biot–Savart Law · Complete trigonometry explained · Special cases: x=0, x=a, x≫a · Interactive visualizations at every step

Final Result
B = μ₀Ia² / 2(x²+a²)3/2
1

Setting Up the Problem

We have a circular loop of radius a carrying current I. The centre is at C. We want to find the magnetic field at point P, which lies on the axis of the loop at distance x from the centre.

Known quantities
a = radius of loop
I = current in loop
x = distance of P from centre C
r = CP = √(x² + a²)
💡 Why is the axis special?

Every point on the loop is at the same distance r = √(x²+a²) from P. This symmetry is the key that makes the integration possible!

2

Geometry — The Triangle LPC

Consider current element Idl at point L on top of the loop. P is on the axis. The triangle LPC is the core geometric structure.

Key geometric facts
1
LP ⊥ dl — The current element dl is tangent to the loop. The line LP (from L to P) is not tangent. So the angle between dl and LP is 90°. This means sin θ = sin 90° = 1 in Biot–Savart!
2
LC ⊥ CP — The radius LC is perpendicular to the axis CP at C. This gives us the right angle at C.
3
LP = MP = √(x²+a²) — All points on loop are equidistant from P (symmetry of circle).
Why dl ⊥ LP?

The current element dl is always tangent to the circle at L. The vector LP is a secant line from L to a point on the axis. Since dl is tangent and the tangent ⊥ radius at L, and LP lies in the plane containing the axis and radius LC, we get dl ⊥ LP. Therefore the angle between dl and r̂ is 90°.

3

Applying Biot–Savart Law

Biot–Savart Law
dB = (μ₀/4π) · I·dl·sinθ / r²
Substituting values
θ = 90° (dl ⊥ LP as proved in Step 2)
So sinθ = sin90° = 1
r = √(x² + a²) (distance from dl to P)
So r² = x² + a²
dB simplifies to
dB = μ₀I·dl / 4π(x²+a²)
sin90°=1 cancels, r² = x²+a²
💡 Key insight

Because θ = 90° for every element dl on the loop, every element contributes the same magnitude dB. This is unique to the circular loop geometry!

4

Resolving dB into Components

The vector dB at point P is perpendicular to the line LP. We resolve it into two components:

Two components of dB
dBₓ = dB sinφ — along x-axis
This component is along the axis (horizontal). Due to symmetry, these add up from all elements → NET CONTRIBUTION
dB_⊥ = dB cosφ — perpendicular to axis
This component is perpendicular (vertical). Components from opposite elements cancel → ZERO NET
sinφ from the triangle LCP
sinφ = LC/LP = a/√(x²+a²)
cosφ = CP/LP = x/√(x²+a²)
From the right-angled triangle LCP (right angle at C)
💡 Why does dB sinφ survive but dB cosφ cancels?

The loop has 180° rotational symmetry. For every element at L on top, there's a matching element at M at the bottom. Their dB cosφ components point in opposite directions (one up, one down) so they cancel. Their dB sinφ components both point along x-axis in the same direction, so they add!

5

Why ⊥ Components Cancel — Visual Proof

This is the most elegant part of the derivation. The loop has perfect circular symmetry. For every current element, there exists a diametrically opposite element that kills the perpendicular component.

Front view of loop
The Cancellation Logic
For element at L (top): dB points in a direction that has component UP (⊥ to axis)
For element at M (bottom, diametrically opposite): dB has component DOWN (⊥ to axis)
Both elements' dB components along axis point in the same direction → they add!
Mathematical statement
∮ dB cosφ = 0 (cancels)
B = ∮ dB sinφ (survives)
💡 Why both sinφ components are in same direction

The current in the loop flows continuously in one direction (say anti-clockwise when viewed from P). So dB from any element always has its sinφ component pointing towards P from the right (towards +x direction). They all add constructively!

6

The Integration — Step by Step

This is where all the pieces come together. We integrate dB·sinφ around the full loop.

A
Start: net B = ∮ dB sinφ
B = ∮ dB sinφ
Integrate over full loop; only sinφ component survives
B
Substitute dB and sinφ
dB = μ₀I·dl / 4π(x²+a²)
sinφ = a / √(x²+a²)
B = ∮ [μ₀I·dl / 4π(x²+a²)] · [a/√(x²+a²)]
C
Pull constants outside integral

x, a, μ₀, I are all constants with respect to dl (they don't change as we go around the loop). So they come out of the integral!

B = (μ₀Ia / 4π) · [1/(x²+a²)^(3/2)] · ∮ dl
[x²+a²]^(3/2) = (x²+a²)·√(x²+a²) — combining the two denominators
D
Evaluate ∮ dl = 2πa (circumference!)

∮ dl means integrate the element length dl over the full closed circular loop. This is simply the circumference = 2πa. Beautiful!

Final Result
B = μ₀Ia² / 2(x²+a²)^(3/2)
(μ₀Ia/4π) · (1/(x²+a²)^(3/2)) · 2πa = μ₀Ia²/2(x²+a²)^(3/2)
7

The Complete Formula — All Forms

Single loop
B = μ₀Ia² / 2(x²+a²)^(3/2)
N=1 turn
N-turn coil
B = μ₀NIa² / 2(x²+a²)^(3/2)
multiply by N turns
At centre (x=0)
B₀ = μ₀I / 2a
x=0, (a²)^(3/2)=a³, B=μ₀I/2a
Direction of B — Right Hand Rule

Curl the fingers of right hand in the direction of current flow in the loop → the thumb points in the direction of B on the axis.

Full Derivation in One Place
1
dB = (μ₀/4π) · Idl·sin90° / (x²+a²) = μ₀Idl / 4π(x²+a²)   ← Biot–Savart, θ=90°
2
sinφ = a/√(x²+a²)   ← from right triangle LCP
3
B = ∮ dB·sinφ   ← cosφ components cancel by symmetry
4
B = (μ₀Ia / 4π(x²+a²)^(3/2)) · ∮dl   ← pull constants out
✓
B = μ₀Ia² / 2(x²+a²)^(3/2)   ← ∮dl = 2πa
8

Special Cases

Case 1
x = 0
Point P is at the centre of the loop.
Substituting x=0:
B = μ₀Ia² / 2(0+a²)^(3/2)
B = μ₀Ia² / 2·a³
B₀ = μ₀I / 2a
Physical meaning

Maximum field. All elements at same distance a from centre. Every dB points along axis.

Case 2
x = a
P is at distance equal to radius from centre.
Substituting x=a:
B = μ₀Ia² / 2(a²+a²)^(3/2)
B = μ₀Ia² / 2(2a²)^(3/2)
B = μ₀Ia² / 2·2√2·a³
B = μ₀I / 4√2·a
Compared to B₀

B(x=a) = B₀/(2√2) ≈ 0.354 × B₀. The field drops to about 35% of centre value.

Case 3
x ≫ a
P is very far from loop — magnetic dipole approximation.
Since x≫a, neglect a²:
(x²+a²)^(3/2) ≈ x³
B ≈ μ₀Ia² / 2x³
Let M = Iπa² (dipole moment):
B ≈ μ₀M / 2πx³
Magnetic dipole!

At large distances B ∝ 1/x³ — same as electric dipole field. The loop acts as a magnetic dipole.

Comparison at a Glance
Condition Formula B/B₀ Physical meaning
x = 0 B₀ = μ₀I/2a 1.000 (max) Centre of loop, maximum field
x = a μ₀I/4√2·a 0.354 Distance = radius, field ≈ 35%
x = 2a μ₀Ia²/2(5a²)^(3/2) 0.089 Double radius, field ≈ 9%
x ≫ a ≈ μ₀Ia²/2x³ → 0 (1/x³) Magnetic dipole regime
