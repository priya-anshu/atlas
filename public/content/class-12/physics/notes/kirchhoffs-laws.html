Kirchhoff's Laws

Theory  ·  Diagrams  ·  Numericals  ·  Beginner to Advanced

⬡
First Law — Junction Rule
Kirchhoff's Current Law (KCL)
At any junction (node) in an electrical circuit, the algebraic sum of all currents meeting at the node is zero. Equivalently, the total current entering a node equals the total current leaving it.
ΣIin = ΣIout     OR     ΣI = 0 (at node)
BASIS: Conservation of Electric Charge
Current entering a node → taken as positive (+)
Current leaving a node → taken as negative (−)
Applies to every node in any network, DC or AC
Charge cannot be created or destroyed at a node
↻
Second Law — Loop Rule
Kirchhoff's Voltage Law (KVL)
The algebraic sum of all EMFs and voltage drops around any closed loop in a circuit is zero. Energy gained by charges from EMF sources equals energy lost in resistances.
ΣE = ΣIR     OR     ΣV = 0 (around loop)
BASIS: Conservation of Energy
Choose a traversal direction (CW or CCW)
EMF traversed − to + → positive; + to − → negative
Resistor in current direction → −IR; against → +IR
A charge completing a loop returns to same potential
±
Quick Reference
Sign Convention Summary
KCL SIGN RULE
Currents IN → +ve
Currents OUT → −ve
ΣI = 0 at every node
KVL SIGN RULE
EMF (−→+) → +E
EMF (+→−) → −E
Resistor (with I) → −IR
Tip: Always assign current directions before applying KVL. If the solved value is negative, it simply means the actual direction is opposite to your assumed direction — the magnitude is still correct.

1
Beginner KCL Theoretical
State KCL and its physical basis
State Kirchhoff's Current Law. What physical principle is it based on? Give a real-world analogy.
▾
◈ QUESTION State Kirchhoff's Current Law. What physical principle is it based on? Give a real-world analogy to help understand it.
1
Statement of KCL
At any junction (node) in an electrical circuit, the algebraic sum of all currents is zero. ΣI = 0 at a node OR: ΣI_in = ΣI_out Convention: Currents entering the node are positive (+); currents leaving are negative (−).
2
Physical Basis
KCL is based on the Law of Conservation of Electric Charge. Electric charge is neither created nor destroyed at a junction. Since current is rate of flow of charge (I = dq/dt), whatever charge flows into a node per unit time must flow out in the same time — giving ΣI = 0.
3
Real-World Analogy
Think of water pipes meeting at a junction (T-joint): • Pipe 1 supplies 10 L/s into the joint • Pipe 2 supplies 5 L/s into the joint • Pipe 3 must drain 15 L/s out of the joint No water is created or destroyed at the joint — it all balances. Exactly the same logic applies to electric current at a circuit node.
2
Beginner KVL Theoretical
State KVL and explain sign convention
State Kirchhoff's Voltage Law. Explain the sign convention for EMF sources and resistors in a loop.
▾
◈ QUESTION State Kirchhoff's Voltage Law. Explain the sign convention used when traversing a closed loop containing EMF sources and resistors. What is its physical basis?
1
Statement of KVL
The algebraic sum of all potential differences (EMFs and resistive drops) around any closed loop in a circuit is zero. ΣE − ΣIR = 0 OR ΣV = 0 (around the loop) This is true for any closed path in any network, regardless of complexity.
2
Sign Convention — EMF Sources
Choose a traversal direction (clockwise or anticlockwise) for the loop. • Traverse EMF from (−) terminal to (+) terminal → take as +E [going with the EMF rise] • Traverse EMF from (+) terminal to (−) terminal → take as −E [going against the EMF] Simple rule: if the EMF "helps" you move in the traversal direction → positive.
3
Sign Convention — Resistors
• Traverse resistor in the SAME direction as assumed current I → voltage DROP = −IR • Traverse resistor OPPOSITE to assumed current I → voltage RISE = +IR Simple rule: moving in the direction of current → you're going "downhill" in potential → negative.
4
Physical Basis
KVL is based on Conservation of Energy. When a unit charge makes a complete loop, it returns to its starting potential. So the net work done on the charge = net work done by the charge → net energy gain = net energy loss → ΣV = 0.

Intermediate KCL Numerical
KCL at a 4-branch node
At node P: I₁=3A and I₂=5A flow in, I₃=4A flows out. Find magnitude and direction of I₄.
▾
◈ QUESTION At a node P, four branches meet. Currents I₁ = 3A and I₂ = 5A flow INTO node P. Current I₃ = 4A flows OUT of P. Using KCL, find the magnitude and direction of I₄.
Circuit Diagram — Node P
1
Apply KCL at Node P
ΣI_in = ΣI_out I₁ + I₂ = I₃ + I₄ Assume I₄ is leaving the node (downward). We will verify from the sign of the answer.
2
Substitute known values
3 + 5 = 4 + I₄ 8 = 4 + I₄ I₄ = 8 − 4 I₄ = 4 A
3
Interpret the result
I₄ = +4 A → positive value confirms our assumed direction is correct. ∴ I₄ = 4A flows OUT of node P (downward in the diagram). Verification: Currents IN = 3 + 5 = 8 A Currents OUT = 4 + 4 = 8 A ✓

Intermediate KVL Numerical
Single-loop series circuit
Series circuit: 12V battery, R₁=2Ω, R₂=4Ω, R₃=6Ω. Find current and voltage across each resistor using KVL.
▾
◈ QUESTION A series circuit has a 12V battery (negligible internal resistance), R₁ = 2Ω, R₂ = 4Ω, and R₃ = 6Ω all connected in series. Using Kirchhoff's Voltage Law, find: (a) The current I flowing in the circuit. (b) The voltage drop across each resistor. (c) Verify your answer using KVL.
Series Loop Circuit
1
Apply KVL (traversing clockwise)
Starting from the negative terminal of the battery, going clockwise: +12 − I·R₁ − I·R₂ − I·R₃ = 0 12 − 2I − 4I − 6I = 0
2
Solve for current I
12 − 12I = 0 12I = 12 I = 1 A ∴ Current in the circuit = 1 A
3
Voltage across each resistor
V₁ = I × R₁ = 1 × 2 = 2 V V₂ = I × R₂ = 1 × 4 = 4 V V₃ = I × R₃ = 1 × 6 = 6 V
4
Verification (KVL check)
Sum of all voltage drops = V₁ + V₂ + V₃ = 2 + 4 + 6 = 12 V This equals the EMF (12V) ✓ KVL confirmed: 12 − 2 − 4 − 6 = 0 ✓

Intermediate KCL + KVL Numerical
Two-mesh circuit — Branch Current Method
E₁=10V, E₂=6V, R₁=2Ω, R₂=4Ω, R₃=3Ω. Find branch currents using KCL and KVL simultaneously.
▾
◈ QUESTION In the two-loop circuit shown: E₁ = 10V, E₂ = 6V, R₁ = 2Ω (left branch), R₂ = 4Ω (shared middle branch), R₃ = 3Ω (right branch). Using Kirchhoff's Laws (KCL + KVL), find the current through each branch. State the equations clearly.
Two-Mesh Circuit
1
Assign Branch Currents
I₁ = current through left branch (E₁ and R₁) I₂ = current through right branch (E₂ and R₃) I₃ = current through middle branch (R₂) KCL at top node: I₁ = I₂ + I₃ → I₃ = I₁ − I₂ ... (KCL)
2
KVL — Loop 1 (Left loop, clockwise)
+E₁ − I₁·R₁ − I₃·R₂ = 0 10 − 2I₁ − 4(I₁ − I₂) = 0 10 − 2I₁ − 4I₁ + 4I₂ = 0 10 = 6I₁ − 4I₂ ... (1)
3
KVL — Loop 2 (Right loop, clockwise)
+E₂ + I₃·R₂ − I₂·R₃ = 0 (Note: traversing R₂ upward against I₃ → +I₃R₂) 6 + 4(I₁ − I₂) − 3I₂ = 0 6 + 4I₁ − 4I₂ − 3I₂ = 0 4I₁ − 7I₂ = −6 ... (2)
4
Solve Simultaneous Equations
Eq (1): 6I₁ − 4I₂ = 10 → multiply by 7: 42I₁ − 28I₂ = 70 Eq (2): 4I₁ − 7I₂ = −6 → multiply by 4: 16I₁ − 28I₂ = −24 Subtract: (42−16)I₁ = 70−(−24) = 94 26I₁ = 94 I₁ = 94/26 ≈ 3.62 A From Eq (1): 6(3.62) − 4I₂ = 10 21.69 − 4I₂ = 10 I₂ = 11.69/4 ≈ 2.92 A
5
Find I₃ and Final Answers
I₃ = I₁ − I₂ = 3.62 − 2.92 = 0.70 A ━━━━━━━━━━━━━━━━━━━━━━━━━━ I₁ ≈ 3.62 A (left branch, downward through E₁) I₂ ≈ 2.92 A (right branch, downward through E₂) I₃ ≈ 0.70 A (middle branch R₂, downward) ━━━━━━━━━━━━━━━━━━━━━━━━━━ Verification (KCL): I₁ = I₂ + I₃ 3.62 ≈ 2.92 + 0.70 = 3.62 ✓

Advanced KCL + KVL Numerical
Unbalanced Wheatstone Bridge
E=20V, r=1Ω, P=3Ω, Q=6Ω, R=6Ω, S=3Ω, G=5Ω. Bridge NOT balanced. Find galvanometer current using Kirchhoff's Laws.
▾
◈ QUESTION In a Wheatstone bridge: battery EMF E = 20V with internal resistance r = 1Ω, P = 3Ω, Q = 6Ω, R = 6Ω, S = 3Ω, galvanometer resistance G = 5Ω. The bridge is NOT balanced. Using Kirchhoff's Laws, find: (a) Verify the bridge is unbalanced. (b) Find the current through the galvanometer.
Wheatstone Bridge Circuit
1
Check Balance Condition
Wheatstone bridge is balanced when: P/Q = R/S P/Q = 3/6 = 0.5 R/S = 6/3 = 2.0 0.5 ≠ 2.0 → Bridge is NOT balanced ✓ Therefore, galvanometer current ≠ 0.
2
Assign Mesh Currents
Use mesh analysis with 3 mesh currents: I₁ = mesh current in left mesh A-B-D (clockwise) I₂ = mesh current in right mesh A-C-D (clockwise) I_G = current through galvanometer B→C KCL at node B: I_G = I₁ − I₁' (handled by mesh method) Total current from battery = I₁ + I₂
3
KVL — Mesh 1 (A→B→D→A, clockwise)
E − r(I₁+I₂) − P·I₁ − G(I₁−I₂) = 0 Substituting values: 20 − 1(I₁+I₂) − 3I₁ − 5(I₁−I₂) = 0 20 − I₁ − I₂ − 3I₁ − 5I₁ + 5I₂ = 0 20 = 9I₁ − 4I₂ ... (1)
4
KVL — Mesh 2 (A→C→D→A, clockwise)
E − r(I₁+I₂) − Q·I₂ + G(I₁−I₂) = 0 (G term is positive because I₁−I₂ opposes mesh 2) 20 − (I₁+I₂) − 6I₂ + 5(I₁−I₂) = 0 20 − I₁ − I₂ − 6I₂ + 5I₁ − 5I₂ = 0 20 = −4I₁ + 13I₂ → 4I₁ − 13I₂ = −20 ... (2)
5
Solve the Equations
Eq (1): 9I₁ − 4I₂ = 20 → ×13: 117I₁ − 52I₂ = 260 Eq (2): 4I₁ − 13I₂ = −20 → ×4: 16I₁ − 52I₂ = −80 Subtract: 101I₁ = 340 I₁ = 340/101 ≈ 3.37 A From (1): 9(3.37) − 4I₂ = 20 30.33 − 4I₂ = 20 I₂ = 10.33/4 ≈ 2.58 A
6
Galvanometer Current
I_G = I₁ − I₂ = 3.37 − 2.58 ━━━━━━━━━━━━━━━━━━━━━━━━━━ I_G ≈ 0.79 A (from B to C through galvanometer) ━━━━━━━━━━━━━━━━━━━━━━━━━━ Note: If bridge were balanced (P/Q = R/S), then I_G = 0. The unbalance causes this 0.79A deflection current.

Advanced KCL + KVL Numerical
Node Voltage Method — 5-resistor network
Two nodes A and B. R₁=5Ω, R₂=10Ω, R₃=20Ω, R₄=4Ω, R₅=8Ω connected to 24V source. Find VA, VB, and all branch currents.
▾
◈ QUESTION In the circuit: R₁=5Ω connects node A to 24V supply. R₂=10Ω connects node A to ground. R₃=20Ω connects node A to node B. R₄=4Ω connects node B to ground. R₅=8Ω connects node B to 24V supply. Using the Node Voltage Method (Kirchhoff's Laws), find: (a) Voltages VA and VB. (b) Current through each resistor. (c) Verify using KCL.
Node Voltage Circuit
1
Set up Node Voltage Equations
Reference node: GND = 0V Unknown voltages: VA, VB At node A — KCL (sum of currents LEAVING node A = 0): (VA − 24)/R₁ + VA/R₂ + (VA − VB)/R₃ = 0 (VA − 24)/5 + VA/10 + (VA − VB)/20 = 0
2
Simplify Node A equation
Multiply through by LCM = 20: 4(VA − 24) + 2VA + (VA − VB) = 0 4VA − 96 + 2VA + VA − VB = 0 7VA − VB = 96 ... (1)
3
Node B equation
KCL at node B (sum of currents LEAVING = 0): (VB − VA)/R₃ + VB/R₄ + (VB − 24)/R₅ = 0 (VB − VA)/20 + VB/4 + (VB − 24)/8 = 0 Multiply by LCM = 40: 2(VB − VA) + 10VB + 5(VB − 24) = 0 2VB − 2VA + 10VB + 5VB − 120 = 0 −2VA + 17VB = 120 ... (2)
4
Solve Equations (1) and (2)
From (1): VB = 7VA − 96 Substitute into (2): −2VA + 17(7VA − 96) = 120 −2VA + 119VA − 1632 = 120 117VA = 1752 VA = 1752/117 = 14.97 V ≈ 15 V VB = 7(15) − 96 = 105 − 96 = 9 V ━━━━━━━━━━━━━━━━━━━━━━━━ VA = 15 V, VB = 9 V ━━━━━━━━━━━━━━━━━━━━━━━━
5
Calculate Branch Currents
I₁ = (24 − VA)/R₁ = (24 − 15)/5 = 9/5 = 1.80 A I₂ = VA/R₂ = 15/10 = 15/10 = 1.50 A I₃ = (VA − VB)/R₃ = (15 − 9)/20 = 6/20 = 0.30 A I₄ = VB/R₄ = 9/4 = 9/4 = 2.25 A I₅ = (24 − VB)/R₅ = (24 − 9)/8 = 15/8 = 1.875 A
6
Verification using KCL
At node A: I₁ = I₂ + I₃ 1.80 = 1.50 + 0.30 = 1.80 ✓ At node B: I₃ + I₅ = I₄ 0.30 + 1.875 = 2.175 ≈ 2.25 (small rounding error due to VA≈15) Exact: VA = 14.974, VB = 8.818 I₄ = 8.818/4 = 2.205 A I₃ = (14.974−8.818)/20 = 0.308 A I₅ = (24−8.818)/8 = 1.898 A I₃ + I₅ = 0.308 + 1.898 = 2.206 ≈ I₄ ✓
